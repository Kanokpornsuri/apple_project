# Apple Stock Data Analysis Pipeline

This project is a modular Python package designed to load, clean, analyze, and visualize Apple (AAPL) historical stock data.

## Project Structure
* `data/` : Contains the raw dataset (`apple.csv`).
* `pipeline/` : Contains modular Python functions (`load.py`, `clean.py`, `analyze.py`, `visualize.py`).
* `outputs/` : Stores the generated data visualization charts.
* `main.py` : The main executable script that runs the end-to-end pipeline.
* `requirements.txt` : Lists all the required Python libraries.
